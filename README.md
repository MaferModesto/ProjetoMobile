O Projeto Mobile é um aplicativo de lista colaborativa desenvolvido utilizando React Native, com foco em praticar conceitos modernos de desenvolvimento mobile e construção de aplicações reais. A proposta do aplicativo é permitir que múltiplos usuários interajam em uma mesma lista, podendo adicionar novos itens de forma simples e intuitiva.
Um dos principais diferenciais do projeto é o controle de permissões implementado: embora todos os usuários autenticados possam adicionar itens à lista, apenas o autor de cada item tem permissão para editá-lo ou excluí-lo. Isso garante maior organização e segurança no uso da aplicação, além de simular regras de negócio comuns em sistemas reais.
O aplicativo conta com um sistema de autenticação integrado ao Firebase, permitindo o cadastro e login de usuários de forma segura. Cada item criado na lista é vinculado ao usuário responsável, possibilitando o controle individual sobre as ações realizadas dentro da aplicação.
Durante o desenvolvimento, foram utilizados conceitos importantes como gerenciamento de estado, navegação entre telas e integração com banco de dados em tempo real. O projeto tem como objetivo principal consolidar conhecimentos em desenvolvimento mobile, além de servir como parte do portfólio, demonstrando habilidades práticas na criação de aplicações completas, com autenticação, regras de negócio e interação entre usuários.
